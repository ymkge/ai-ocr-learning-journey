# AI OCR Learning Journey & Portfolio

AI OCR（コンピュータビジョン × 自然言語処理）の仕組みを基礎から学び、実務で通用するAIエンジニア兼データサイエンティストとしてのスキルを習得するための学習リポジトリです。

単に既存のAPIを呼び出すだけでなく、画像のインプット最適化、前処理（OpenCV）、テキスト検出・認識の内部アルゴリズムの理解、そしてデータサイエンスに基づく精度評価と後処理（NLP）までの一連のパイプラインを体系的に構築・実験していきます。

---

- [x] **Phase 1: 基礎環境構築と「動かす」体験**
  - [x] GitHub環境整備・Python環境構築
  - [x] 既存OSS（EasyOCRなど）を用いた簡易OCRスクリプトの実装

  - [ ] 評価指標（CER: 文字エラー率、WER: 単語エラー率、編集距離）の自作実装

## 👤 このリポジトリについて

- **目的**: AI OCRの技術スタックを通じて、コンピュータビジョンと自然言語処理が融合した領域の実践スキルを磨く。
- **目指す姿**: データの特性を理解し、アルゴリズムの選定からシステム化、そして継続的な精度改善までを主導できるAIエンジニア兼データサイエンティスト。
